# 첫 활동이네요!!

앞으로 열심히 해보겠습니다.

샘플 파이썬 코드입니다.

``` python
def is_palindrome(word: str) -> bool:
  if len(word) <= 1:
    return True
  elif word[0:1] != word[len(word) - 1:len(word)]:
    return False
  else:
    return is_palindrome(word[1:len(word) - 1])
```
